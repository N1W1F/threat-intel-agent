🧭 Orchestrator

ينسّق التتابع بين الوكلاء ويجمع النتائج.

🗂️ Asset Auditor

يقرأ البرامج المثبتة عبر winget — قراءة فقط.

🛰️ Threat Hunter

يبحث ثغرات CVE من NVD — شبكة فقط، مضيف ثابت.

🚨 KEV Checker

يطابق قائمة CISA للثغرات المستغلة فعلياً بالعالم الحقيقي.

⚖️ Decision Agent

يصنّف عاجل/روتيني، ويحسب مؤشر الصحة — صفر صلاحية تنفيذ.

📝 Remediation

يكتب التقرير النهائي — مسار ملف ثابت واحد فقط.

🔄 Package Manager

يفحص ويطبّق تحديثات winget — يرفض أي معرّف غير مفحوص.

🤖 Analyst (LLM)

يشرح، يجاوب أسئلتك، يعيد التقييم — عبر Ollama محلي.
